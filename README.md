📊 Home Broker Pro

Um simulador de home broker com gráficos em tempo real, inspirado em plataformas de trading.

🚀 Funcionalidades:

📈 Gráfico de linha e candlestick
📊 Atualização automática de preços
📉 Lista de ações simuladas
⚙️ Controle de intervalo de atualização

🛠️ Tecnologias:

HTML
CSS
JavaScript
Chart.js
Python

▶️ Como usar
Baixe ou clone o projeto
Abra o arquivo index.html no navegador
Pronto, o sistema já funciona
